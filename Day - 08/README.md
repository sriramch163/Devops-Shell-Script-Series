## Day 8 – Server Reachability Checker

This script checks whether multiple servers or domains are reachable using ping.

Why this is useful:
Helps verify network connectivity between systems and detect unreachable hosts.

Usage:
./day-08-server-ping-check.sh



Example Output
🔍 Checking server connectivity...
-----------------------------------
google.com is reachable
github.com is reachable
8.8.8.8 is reachable
localhost is not reachable
