## Day 6 – Folder Size Analyzer

This script shows folder sizes inside a given directory and sorts them from largest to smallest.

Why this is useful:
Helps identify which directory is consuming disk space during outages.

Usage:
./day-06-folder-size.sh <directory-path>

Example Output
🔍 Analyzing folder sizes in: /var
------------------------------------------
2.3G    /var/lib
1.2G    /var/log
450M    /var/cache
20M     /var/tmp
