## Day 7 – System Health Report

This script provides a quick overview of system health including CPU, memory, and disk usage.

Why this is useful:
Helps DevOps engineers quickly diagnose server performance issues.

Usage:
./day-07-system-health.sh

Example Output
==============================
   System Health Report
==============================

CPU Usage:
12.5%

Memory Usage:
              total   used   free
Mem:           7.6G   3.1G   4.5G

Disk Usage:
Filesystem      Size  Used  Avail  Use%
/dev/sda1        50G   22G    25G   47%

Health check completed
